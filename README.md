# How to start using Svaeva SDK

## install poetry

```bash
curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python -
```

## Start or enveronment

```bash
poetry shell
```

## Installation

```bash
poetry add git+https://github.com/Daisie-Bell/svaeva-sdk.git
```
